# 项目说明

## 基本信息

- 项目名称：`target-pdf-compress`
- 产品域名：`filesmaller.space`
- 域名购买平台：Namecheap
- 代码仓库：`https://github.com/horaceace/target-pdf-compress`
- 本地目录：`/Users/chenshaojian/AI创业尝试工作空间/products/出海网站项目/target-pdf-compress`

## 部署信息

- 生产部署平台：Cloudflare Workers
- 当前 Workers 地址：`https://target-pdf-compress.jiansay-java.workers.dev`
- 当前部署方式：GitHub Actions 自动部署
- 触发条件：推送到 `master` 分支后自动发布

## 域名与解析

- 域名在 Namecheap 购买
- 域名托管和解析在 Cloudflare
- 站点主域名建议使用：`filesmaller.space`
- `www.filesmaller.space` 需要单独配置
  - 方案一：绑定为同一个 Worker 自定义域名
  - 方案二：在 Cloudflare Rules 中 301 跳转到 `filesmaller.space`

## 技术栈

- Next.js `15.5.18`
- React `19.1.0`
- TypeScript
- `pdf-lib`
- `@opennextjs/cloudflare`
- `wrangler`

## 当前功能

- 首页工具导航
- 多个 PDF 压缩场景页
- 浏览器内上传 PDF
- 浏览器内执行基础压缩
- 显示压缩前后体积
- 下载压缩后的 PDF

## 代码位置

主要目录：

- `app/`：页面与路由
- `components/`：页面组件与上传组件
- `content/`：场景页文案配置
- `lib/pdf/`：PDF 压缩逻辑
- `.github/workflows/`：Cloudflare 自动部署流程
- `wrangler.jsonc`：Workers 配置
- `open-next.config.ts`：OpenNext Cloudflare 配置

## 后续维护
