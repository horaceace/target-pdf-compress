# target-pdf-compress

First-pass Next.js project scaffold for the outbound PDF target-size compression product.

## Run locally

```bash
npm install
npm run dev
```

## Included in this scaffold

- Home page
- Dynamic SEO landing pages for five target-size routes
- Privacy page
- Shared content config in `content/tool-pages.ts`
- Shared page template in `components/tool-page-template.tsx`

## Not included yet

- Real PDF compression logic
- Analytics
- Deployment config
- ShipAny integration
